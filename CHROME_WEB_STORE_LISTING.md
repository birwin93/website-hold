## Short Description (132 chars max)

Pause before distracting sites with a confirmation screen so you can choose intentionally before continuing.

## Detailed Description

Hold Up adds a deliberate pause before opening websites you choose.

Add one or more domains in the extension options (for example, `youtube.com` or `instagram.com`). When a tab tries to open one of those websites, Hold Up shows an interstitial confirmation screen instead of loading the page immediately.

From that screen, you can:

- Continue: open the requested page
- Go back: return without opening it

Hold Up is designed for focus and intentional browsing. It does not block your access permanently; it simply adds a conscious decision point before visiting configured websites.

## Single Purpose Description

Hold Up helps users browse more intentionally by showing a confirmation interstitial before loading user-configured websites.

## Permission Justification

## Reviewer Test Instructions

1. Install extension.
2. Open Extension options.
3. Add `example.com` and save.
4. Navigate to `https://example.com` in a tab.
5. Verify interstitial appears with “Hold up there”.
6. Click Go back and confirm navigation is canceled.
7. Navigate again to `https://example.com`.
8. Click Continue and confirm page opens.
